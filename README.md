# Python

Lab work for a semester-long Python course (AGH), covering the language
from basic syntax through OOP, functional programming, file I/O,
decorators, dataclasses, pytest, and CPython C extensions.

Each `labNN/` folder is self-contained and documents itself in its own
`README.md`: the task requirements and the files in the folder.

## Labs

| Lab | Topic |
|---|---|
| [lab01](lab01/) | Introduction to Python |
| [lab02](lab02/) | Command-line arguments, list comprehensions, shallow copies |
| [lab03](lab03/) | String-only path manipulation |
| [lab04](lab04/) | Dictionaries, dict comprehensions, permutations |
| [lab05](lab05/) | Sensor simulation (mutable defaults & variadic arguments) |
| [lab06](lab06/) | Functional programming: map/filter/reduce/lambda |

| [lab12](lab12/) | Decorators for HTTP request monitoring |
| [lab13](lab13/) | Dataclasses and structural pattern matching |
| [lab14](lab14/) | Writing a Python C extension |
